# Cloud Gephi (Dev) Project

Cloud Gephi is developed as part as a Google Summer of Code (GSoC) project.

This project proposes to bring some of Gephi's features to the cloud and build a simple yet robust network gallery on the web.

Student: Romain Yon

Mentor: Mathieu Bastian
